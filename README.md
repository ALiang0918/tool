# 🔧 工具箱

开发中常用的几个纯前端小工具，零依赖，浏览器直接打开即可使用。

## 工具列表

| 工具 | 说明 |
|------|------|
| [Java 字段对比器](./java-compare.html) | 对比两个 Java 对象的属性差异，快速定位共有字段、类型冲突、独有字段 |
| [SQL IN 条件格式化](./sql-formatter.html) | 粘贴多行数据，一键生成逗号分隔的 IN 条件，支持加引号、去重、排序、多行/压缩一行 |

## 使用方式

直接打开 `index.html` 进入导航首页，或打开对应工具的 HTML 文件。
